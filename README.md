# AI_Agent-releases

Solo `version.json` per il controllo aggiornamenti della CLI TrueNex Agent.

- **version**: versione corrente (es. 0.3.0)
- **force_update**: se `true`, la CLI propone l’aggiornamento anche senza cambio versione
- **update_full**: se `true`, l’aggiornamento esegue anche `pip install -r requirements.txt` (reinstall completo); se `false` o assente, solo `git pull` + `pip install -e .` (rapido)

URL raw per la CLI: `https://raw.githubusercontent.com/marcomnit/AI_Agent-releases/main/version.json`
